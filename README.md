# Data-tructures-and-algorithms

This repository contains laboratory work from the course 
**Data Structures and Algorithms**.

The course focused on fundamental data structures, algorithm design,
recursion, complexity analysis, and problem-solving strategies.

---

## Topics Covered

### Chapter 01 – Arrays, Simple Sorting, Basics
- Array manipulation
- Basic sorting algorithms
- Problem-solving fundamentals

### Chapter 02 – Recursion
- Recursive functions
- Divide-and-conquer thinking
- Recursive problem solving

### Chapter 03 – Time Complexity
- Big-O notation
- Algorithm efficiency analysis
- Comparing algorithm performance

### Chapter 04 – Struct and Class
- Object-oriented programming
- Custom data types
- Encapsulation and abstraction

### Chapter 05 – Queues and Applications
- Queue implementation
- FIFO structures
- Practical use cases

### Chapter 06 – Trees
- Tree data structures
- Traversals (DFS, BFS)
- Binary trees

### Chapter 07 – Backtracking
- Recursive search
- Constraint solving
- State-space exploration

### Chapter 08 – Heap
- Priority queues
- Heap operations
- Heap sort

### Chapter 11 – Graph
- Graph representations
- BFS & DFS
- Shortest path basics

### Chapter 12 – Sorting
- Advanced sorting algorithms
- Time complexity comparison
- Optimization strategies
